# DNS 모듈 설명

---

## 1) 모듈 역할

- Route53 Hosted Zone을 생성(또는 기존 존 참조)한다.
- 필요 시 ACM 인증서를 발급하고 DNS 검증 레코드를 만든다.

---

## 2) .tf 파일 구성

### main.tf

- `aws_route53_zone`로 Hosted Zone을 생성한다.
- `aws_acm_certificate`로 인증서를 요청한다.
- `aws_route53_record`로 DNS 검증 레코드를 만든다.
- `aws_acm_certificate_validation`으로 인증서를 검증한다.

### variables.tf

- `domain_name`: 대상 도메인.
- `create_hosted_zone`: Hosted Zone 신규 생성 여부.
- `enable_acm`: ACM 인증서 발급 여부.
- `acm_subject_alternative_names`: 인증서 SAN 목록.

### outputs.tf

- `hosted_zone_id`: Hosted Zone ID.
- `hosted_zone_name_servers`: 네임서버 목록.
- `acm_certificate_arn`: 인증서 ARN.

---

# 빠른 요약

- Route53 Hosted Zone과 ACM 인증서를 구성한다.
- DNS 검증 레코드까지 자동으로 생성한다.
