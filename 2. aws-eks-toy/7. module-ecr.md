# ECR 모듈 설명

---

## 1) 모듈 역할

- Backend/Frontend용 ECR 리포지토리를 생성한다.
- 이미지 태그를 Mutable로 설정한다.

---

## 2) .tf 파일 구성

### main.tf

- `aws_ecr_repository`로 Node/React 리포지토리를 생성한다.
- 스캔 설정은 기본적으로 비활성화한다.

### variables.tf

- `nodejs_repo_name`: Node.js 리포 이름.
- `react_repo_name`: React 리포 이름.

### outputs.tf

- `nodejs_repo_url`: Node 리포 URL.
- `react_repo_url`: React 리포 URL.

---

# 빠른 요약

- ECR 리포를 만들어 이미지 저장소를 확보한다.
- 리포 URL을 출력해 배포 설정에 사용한다.
